# Sacred Geometry Generator

Interactive sacred geometry pattern generator — free browser tool by [@ChanneledCode](https://x.com/ChanneledCode).

🔗 **Live:** https://channeledcode.github.io/sacred-geometry-generator

## Patterns
- **Flower of Life** — overlapping circles encoding all platonic solids
- **Seed of Life** — seven circles, the genesis pattern
- **Metatron's Cube** — 13 circles, all platonic solids contained within
- **Vesica Piscis** — two circles, √3 ratio, the womb of geometry
- **Golden Spiral** — Fibonacci sequence made visible
- **Sri Yantra** — nine interlocking triangles, Tantric tradition

## Features
- Parametric controls: scale, complexity, colors, line width
- Download as SVG or PNG (1200×1200)
- Copy embed code
- Zero dependencies — pure vanilla JS

## Stack
Single `index.html` file. GitHub Pages compatible.
